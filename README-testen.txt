Testversie website de Rederij

Wat heb je nodig
- Een Windows laptop of pc
- Node.js geinstalleerd

Zo start je de website
1. Pak het zipbestand uit in een eigen map.
2. Dubbelklik op `start-website.bat`
3. Er opent een zwart venster.
4. Open daarna in je browser:
   http://127.0.0.1:3000

Als dubbelklikken niet werkt
1. Open de uitgepakte map
2. Klik in de adresbalk van Verkenner
3. Typ `powershell` en druk op Enter
4. Typ daarna:
   node server.js

Belangrijk
- De testdata staat in de map `data`
- Alles draait lokaal op je eigen laptop
- Wijzigingen die jij maakt blijven alleen in jouw eigen kopie staan

Belangrijke pagina's
- Home: http://127.0.0.1:3000
- Planning: http://127.0.0.1:3000/planning
- Zeildagen: http://127.0.0.1:3000/zeildagen
- Logboek: http://127.0.0.1:3000/logboek
- Kasboek: http://127.0.0.1:3000/kasboek
- Beheer: http://127.0.0.1:3000/beheer

Beheer
- De eerste keer kun je via `/beheer` een beheeraccount aanmaken
- Daarna kun je daar inloggen
